# Shengen stay planner
A simple web page that allows users to calculate how many days he/she have in the Schengen zone based on the rule 90 days per 180. It is not an official app.
Similar to https://ec.europa.eu/assets/home/visa-calculator/calculator.htm, but with nicer UI

## Implementation details
Pure HTML/CCS/JS. Use Claude Code

At first, it will be rolled out on GitHub
